OneDrive Auto-Sync
Auto-sync writes directly to a file in your OneDrive on every save. It requires a one-time Azure App registration (free). Follow the setup guide below.
Setup
  1. Go to portal.azure.com → App registrations → New registration
  2. Name it "Budget Tracker", select "Personal Microsoft accounts only"
  3. Under Redirect URIs add: https://YOUR-GITHUB-PAGES-URL
  4. Copy the Application (client) ID
  5. Under API permissions add: Files.ReadWrite (delegated)
  6. Paste your Client ID below and click Connect
Data management
All data is stored locally in your browser's localStorage. Nobody else can access it.
